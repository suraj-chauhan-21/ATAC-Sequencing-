# ATAC-Sequencing-
ATAC-seq: “Who left the DNA doors open?” The setting  Imagine your genome as a huge apartment building.  Each room = a gene  Locked doors = gene OFF  Open doors = gene ON or ready to be ON  The problem: we cannot directly see which doors are open.  So we use a special molecular spy. The key player: Tn5 transposase Tn5
